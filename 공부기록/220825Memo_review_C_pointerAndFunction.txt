* 오늘의 문제1(배열의 원소 출력하는 함수 만들기)
	매개변수에 전달된 배열의 모든 원소들을 첫번째 원소부터 차례대로 출력하는 print_array함수를 정의
	main()함수에서 print_array함수를 이용해 배열 numbers의 모든 원소 출력
	int number[] = { 32, 21, 17, 3, 9, 12, 28, 19, 7, 25 };

* 오늘의 문제2(배열 복사하는 함수 만들기)
	매개변수에 전달된 배열을 다른 배열에 복사하는 copy_array함수를 정의
	main()함수에서 copy_array함수를 이용해 배열 numbers를 배열 copy에 복사하고 배열 copy의 모든 원소 출력
	int number[] = { 32, 21, 17, 3, 9, 12, 28, 19, 7, 25 };
	int copy[] = { 0 };

* 오늘의 문제3(문자열 길이구하는 함수 만들기)
	매개변수에 전달된 문자열의 길이를 구하는 str_len함수 정의
	main함수에서 사용자에게 문자열 S를 입력받고 str_len함수 이용해 문자열 S길이 출력
	(이때, 문자열의 최대 길이는 80으로 제한)
